## Acerca de este proyecto

Este proyecto es un fork de [preguntas-entrevista-react](https://github.com/midudev/preguntas-entrevista-react) creado por [midudev](https://midu.dev/).

El objetivo es adaptar las preguntas de entrevista para que se centren en **Go (Golang)** en lugar de React. Las preguntas cubren temas como goroutines, channels, defer, mutex, interfaces, y más conceptos fundamentales del lenguaje.

## Contribuir

Este es un fork del proyecto original. Si quieres contribuir, puedes hacerlo en el [repositorio original de midudev](https://github.com/midudev/preguntas-entrevista-react) o crear tu propio fork para preguntas de Go.
